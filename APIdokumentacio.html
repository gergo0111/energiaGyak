API Dokumentáció

Alap URL: http://localhost:3000

1. GET /

Egyszerű tesztvégpont, hogy a szerver működik-e.

Kérés

GET /

Válasz

200 OK

Működik a szerver

2. GET /users

Az összes felhasználó lekérése.

Kérés

GET /users

Válasz

200 OK

[
  {
    "id": 1,
    "username": "jdoe",
    "email": "jdoe@example.com",
    "full_name": "John Doe",
    "created_at": "2024-01-01T10:00:00.000Z",
    "updated_at": "2024-01-02T10:00:00.000Z",
    "is_active": true
  },
  ...
]

Hiba

500 Internal Server Error

{
  "error": "Szerverhiba történt"
}

3. POST /users

Új felhasználó létrehozása.

Kérés

POST /users
Content-Type: application/json

{
  "username": "jdoe",
  "email": "jdoe@example.com",
  "full_name": "John Doe",
  "is_active": true   // opcionális, alapértelmezett: true
}

Válasz

201 Created

{
  "message": "Felhasználó sikeresen létrehozva",
  "userId": 123
}

Hibák

400 Bad Request (hiányzó mezők esetén)

{
  "error": "Hiányzó kötelező mezők"
}

500 Internal Server Error

{
  "error": "Szerverhiba történt"
}

4. DELETE /users/:id

Felhasználó törlése azonosító alapján.

Kérés

DELETE /users/123

Válasz

200 OK

{
  "message": "Felhasználó sikeresen törölve"
}

Hibák

404 Not Found (ha nincs ilyen felhasználó)

{
  "error": "Felhasználó nem található"
}

500 Internal Server Error

{
  "error": "Szerverhiba történt"
}

5. PUT /users/:id

Felhasználó teljes frissítése azonosító alapján.

Kérés

PUT /users/123
Content-Type: application/json

{
  "username": "ujfelhasznalo",
  "email": "ujemail@example.com",
  "full_name": "Új Név",
  "is_active": false
}

Megjegyzés: Minden mező kötelező.

Válasz

200 OK

{
  "message": "Felhasználó sikeresen frissítve"
}

Hibák

400 Bad Request (hiányzó mezők)

{
  "error": "Hiányzó kötelező mezők: username, email, full_name, is_active"
}

404 Not Found (ha nincs ilyen felhasználó)

{
  "error": "Felhasználó nem található"
}

500 Internal Server Error

{
  "error": "Szerverhiba történt"
}

HTTP Státuszkódok összefoglalója

Kód Jelentés Használat
200 OK Sikeres lekérés, törlés vagy módosítás
201 Created Sikeres létrehozás (POST)
400 Bad Request Hibás vagy hiányzó kérés adatok
404 Not Found Nincs találat (pl. törlés vagy módosítás esetén)
500 Internal Server Error Szerver oldali hiba
